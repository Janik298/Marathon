# Athletic Performance Tracker MVP

This folder contains a first version of an iOS workout tracker built with SwiftUI and SwiftData.

Included features:

- six week training blocks
- exercise database model
- strength workout logging
- cardio workout logging
- weekly review model
- block review model
- local first persistence
- CSV export service

Open the folder in Xcode and build with an iOS 17 or newer target.

Note: this environment does not include Xcode or Swift, so the code still needs to be compiled and tested on a Mac.
